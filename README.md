# Dashboard Hub - Dashboard 聚合中心

一个简洁高效的 Dashboard 聚合工具，帮助你集中管理多个网站的 Dashboard。

## 功能特性

- **多网站聚合** - 将所有常用 Dashboard 集中在一个页面查看
- **标签分类** - 使用标签对 Dashboard 进行分类管理
- **登录管理** - 支持保存登录凭证，方便快速访问需要认证的页面
- **筛选搜索** - 快速搜索和筛选你需要的 Dashboard
- **全屏查看** - 支持全屏模式，获得更好的视觉体验
- **导入/导出** - 轻松备份和迁移你的配置
- **本地存储** - 所有数据保存在本地浏览器，安全可靠

## 使用方法

1. 双击打开 `dashboard-hub.html` 文件
2. 点击右上角「+ 添加 Dashboard」
3. 填写网站名称和 URL
4. 如果需要登录，勾选"需要登录"并保存凭证
5. 点击保存，即可开始使用

## 登录辅助

对于需要登录的网站，Dashboard Hub 提供：
- 登录助手面板 - 快速填入凭证
- 复制凭证功能 - 方便手动登录
- 新标签页打开 - 在新窗口中完成登录

## 数据安全

- 所有数据存储在浏览器本地 (localStorage)
- 不会上传到任何服务器
- 支持导出配置进行备份

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + K` | 快速定位搜索框 |
| `Esc` | 关闭弹窗/退出全屏 |

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无需任何依赖
- 支持所有现代浏览器
